# HackNYU-2025
Win or Crash!
Collaborators: Ivan Ng, Steven Zhu, William Zheng, Zixuan Yu

## How to run
Using the terminal, type "cd win-or-crash", then type "npm install", and finally, "npm run dev". For the commands to run properly, it is recommended to have the latest version of node. After running the commands, there will be a message on where to find the website. Use the url provided to play the game!
